NVM-Windows #

node.js版本管理器 - 符合 POSIX 的 bash 脚本，用于管理多个活动的node.js版本

NVM-Windows 命令 #

命令含义
nvm arch显示node运行在32位还是64位模式。
nvm install <version> [arch]版本可以是node.js版本，也可以是最新稳定版本的“latest”。
nvm list [available]列出node.js的安装。在最后输入“available”，看看可以安装什么。
nvm on启用node.js版本管理。
nvm off禁用node.js版本管理。
nvm proxy [url]设置用于下载的代理。将[url]留空以查看当前代理。[url]设置为“没有”删除代理。
nvm node_mirror [url]设置node.js镜像。默认为https://nodejs.org/dist/。保留[url]为空，使用默认url。
nvm npm_mirror [url]设置npm镜像。默认为https://github.com/npm/cli/archive/。将[url]留空为默认url。
nvm uninstall <version>版本必须是特定的版本。
nvm use [version] [arch]切换到使用指定的版本。可选地指定32/64位架构。nvm使用 <arch> 将继续使用所选版本, 但切换到32/64位模式。
nvm root [path]设置nvm应该存储不同版本node.js的目录。如果 <path> 没有设置，则显示当前的根目录。
nvm version显示当前运行的Windows nvm版本。别名为v。
