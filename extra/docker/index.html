Docker #

image/镜像 #

  • docker imagers 查看已下载的镜像
  • docker pull 拉取镜像
  • docker rmi/image rm 删除镜像

container/容器 #

  • docker ps 查看运行中的容器,-a 查看所有容器
  • docker run 运行镜像生成容器
  • docker stop 停止容器
  • docker rm 删除容器

容器日志 #

  • docker logs --tail=100 <CONTAINER_ID/容器 ID>

选项 #

  • --details 显示更多的信息
  • -f, --follow 跟踪实时日志
  • --since string 显示自某个 timestamp 之后的日志，或相对时间，如 42m（即 42 分钟）
  • --tail string 从日志末尾显示多少行日志， 默认是 all
  • -t, --timestamps 显示时间戳
  • --until string 显示自某个 timestamp 之前的日志，或相对时间，如 42m（即 42 分钟）

其他 #

  • docker -v 查看版本

  • docker login localhost:8080 -u name -p pwd 登录到自托管注册表

  • docker logout ocalhost:8080 登出

常用命令 #

  • docker -v 查看版本
  • docker imager 产看已下载的镜像
  • docker pull 拉取镜像
  • docker rmi/image rm 删除镜像
  • docker run 运行镜像生成容器
  • docker ps 查看运行中的容器,-a 查看所有容器
  • docker stop 停止容器
  • docker restart 重启容器(运行中会重启，关闭中会开启)
  • docker rm 删除容器
  • docker build 用 Dockerfile 文件构建镜像 -t 容器名 -f Dockerfile 文件位置
  • docker attach 容器 ID 进入 Docker 容器
  • sudo docker exec -it “容器 ID” /bin/bash<使用bash> 进入 Docker 容器
  • docker save 保存的是镜像（image）
  • docker export 保存的是容器（container）；
  • docker load 用来载入镜像包
  • docker import 用来载入容器包，但两者都会恢复为镜像；
  • docker load 不能对载入的镜像重命名 而 docker import 可以为镜像指定新名称。
